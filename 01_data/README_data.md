## Raw Data

### `raw/price/wti_hourly_raw.csv`

**Source:** Yahoo Finance API via `yfinance` (ticker: `CL=F`)  
**Generated by:** `02_notebooks/01_yfinance_prices.ipynb`  
**Coverage:** 2 years of hourly data (~11,200 records)

| Column | Type | Description |
| --- | --- | --- |
| `Datetime` | datetime (UTC, index) | Timestamp of the trading hour |
| `Open` | float | Price at the opening of the hour |
| `High` | float | Highest price within the hour |
| `Low` | float | Lowest price within the hour |
| `Close` | float | Price at the close of the hour |
| `Volume` | int | Number of contracts traded within the hour |
| `log_volume` | float | Natural log of Volume — normalizes right-skewed distribution |
| `price_range` | float | High − Low — intraday volatility proxy (Parkinson, 1980) |
| `log_return` | float | log(Close_t / Close_{t-1}) — hourly log return |
| `amihud` | float | \|log_return\| / Volume — illiquidity ratio (Amihud, 2002) |

---

### `raw/news/gdelt_wti_raw.csv`

**Source:** GDELT Project API v2  
**Generated by:** `02_notebooks/03_gdelt_headlines.ipynb`  
**Coverage:** ~4,599 articles across 44 weeks (2024–2026), multiple languages

| Column | Type | Description |
| --- | --- | --- |
| `url` | string | Full URL of the article |
| `url_mobile` | string | Mobile version URL |
| `title` | string | Article headline as published |
| `seendate` | string | Publication timestamp in GDELT format (YYYYMMDDTHHmmSSZ) |
| `socialimage` | string | URL of the article's social media preview image |
| `domain` | string | Source domain (e.g. fxstreet.com, oilprice.com) |
| `language` | string | Detected language of the article |
| `sourcecountry` | string | Country of origin of the publication |
| `week_start` | string | Start date of the download window used to retrieve this article |

---

### `raw/macro/eia_inventories_raw.csv`

**Source:** U.S. Energy Information Administration (EIA) public API  
**Generated by:** `02_notebooks/02_eia_inventories.ipynb`  
**Coverage:** 321 weekly observations (2020–2026)  
**Series:** WCRSTUS1 — U.S. Ending Stocks of Crude Oil excl. Strategic Petroleum Reserve

| Column | Type | Description |
| --- | --- | --- |
| `date` | date | Publication date of the EIA weekly report |
| `inventory_mbbl` | int | U.S. commercial crude oil inventory level in thousand barrels |
| `inventory_change` | float | Week-on-week change in inventory level (this week − last week) |
| `news_direction` | string | Event classification: `bearish` (inventories up), `bullish` (inventories down), `neutral` |
| `datetime_et` | datetime (US/Eastern) | Exact event timestamp — every Wednesday at 10:30 AM ET |

## Processed Data

### `processed/gdelt_wti_clean.csv`

**Generated by:** `02_notebooks/03_gdelt_headlines.ipynb`  
**Records:** ~3,290 articles  
**Description:** Filtered and cleaned version of `gdelt_wti_raw.csv`. English-language articles only, duplicates removed, timestamps parsed to datetime format.

| Column | Type | Description |
| --- | --- | --- |
| `title` | string | Article headline |
| `datetime` | datetime (UTC) | Publication timestamp |
| `domain` | string | Source domain |
| `url` | string | Full article URL |

---

### `processed/gdelt_wti_with_body.csv`

**Generated by:** `02_notebooks/04_gdelt_scraper.ipynb`  
**Records:** ~3,290 articles  
**Description:** Extends `gdelt_wti_clean.csv` with scraped body text. For each article, the first three substantive paragraphs (>50 characters each) are extracted from the article's HTML. Articles behind paywalls or rendered with JavaScript return empty or error strings.

| Column | Type | Description |
| --- | --- | --- |
| `title` | string | Article headline |
| `datetime` | datetime (UTC) | Publication timestamp |
| `domain` | string | Source domain |
| `url` | string | Full article URL |
| `body` | string | First 3 paragraphs of article body text. Empty string or `ERROR:...` if scraping failed |

---

### `processed/gdelt_wti_aligned.csv`

**Generated by:** `02_notebooks/05_alignment.ipynb`  
**Records:** ~2,323 articles  
**Description:** Central modeling dataset for the thesis. Each news article is matched to the closest trading hour in the yfinance dataset using timestamp rounding. Articles published outside NYMEX trading hours (nights, weekends) are discarded. The result merges article metadata with contemporaneous liquidity metrics.

| Column | Type | Description |
| --- | --- | --- |
| `title` | string | Article headline |
| `datetime` | datetime (UTC) | Original article publication timestamp |
| `domain` | string | Source domain |
| `url` | string | Full article URL |
| `datetime_hour` | datetime (UTC) | Publication timestamp rounded to the nearest hour |
| `Close` | float | WTI closing price at the matched trading hour |
| `Volume` | int | Trading volume at the matched trading hour |
| `log_volume` | float | Log-transformed volume — primary liquidity variable |
| `price_range` | float | High − Low at the matched trading hour |
| `log_return` | float | Hourly log return at the matched trading hour |
| `amihud` | float | Amihud illiquidity ratio at the matched trading hour |

## Notes

- **Raw data should never be modified.** All transformations are performed in notebooks and saved to `processed/`.
- **Data files are excluded from version control** (see `.gitignore`). To reproduce the datasets, run the notebooks in order: `01` → `02` → `03` → `04` → `05`.
- **Timezone convention:** all timestamps are stored in UTC unless otherwise noted. The EIA `datetime_et` column is the only exception, stored in `US/Eastern` to reflect the official publication timezone.
